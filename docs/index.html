INTRODUCTION TO CCSCROLL.JS

ccscroll.js is a open-source JavaScript framework or script file that is capable of calculating left (X) values relative to the current scrolling height of the HTML document, and it makes users' lives easier with HTML5 data tags (datasets). Using ccscroll, one can implement their own parallax scrolling without writing a single line of JavaScript code.


FEATURES OF CCSCROLL.JS

  • Can be attached using data tags to elements (HTML5)
  • Higher FPS rate (~60 fps)
  • Multi-directional / Multi-destinational / Multi-starting point configurations
    1. To Left Click here for example
    2. To Right Click here for example
    3. To Center (from left) Click here for example
    4. To Center (from right) Click here for example
  • Pivoting to middle or left
  • Position percentage scaling factor (data-factor, 1 is default)

HOW TO USE CCSCROLL.JS

I use this stylesheet in all of the below examples,

		.background { width:100%; height:250px; background-color:var(--main-primary-color); } 
.content-handle { background:url(res/js.jpg); background-size:100%; width:100px; height:100px; position:relative; top:50%; transform:translateY(-50%); }

From Left to Right:

The Child is moves form left to right, 
	<div class="background" data-scroll="parent">
    <div class="content-handle" data-scroll="child" data-to="right"></div>
</div>

From Right to Left:

The Child is moves form right to left, 
	<div class="background" data-scroll="parent">
    <div class="content-handle" data-scroll="child" data-to="left"></div>
</div>

From Left to Center:

The Child is moves to the center from left, 
	<div class="background" data-scroll="parent">
    <div class="content-handle" data-scroll="child" data-from="left" data-to="center"></div>
</div>

From Right to Center:

The Child is moves to the center from right, 
	<div class="background" data-scroll="parent">
    <div class="content-handle" data-scroll="child" data-from="right" data-to="center"></div>
</div>

Different Speed Factors:

Speed Factor is a data attribute of the child element in the scrolling context. Speed factor or data-factor is used to change the effect on percentage reduction from the vertical scrolled percentage. 
	<div class="background" data-scroll="parent">
    <div class="content-handle" data-scroll="child" data-to="right" data-factor="1"></div>
</div>
<div class="background" data-scroll="parent">
    <div class="content-handle" data-scroll="child" data-to="right" data-factor="0.75"></div>
</div>
<div class="background" data-scroll="parent">
    <div class="content-handle" data-scroll="child" data-to="right" data-factor="0.5"></div>
</div>
<div class="background" data-scroll="parent">
    <div class="content-handle" data-scroll="child" data-to="right" data-factor="0.25"></div>
</div>
<div class="background" data-scroll="parent">
    <div class="content-handle" data-scroll="child" data-to="right" data-factor="0.1"></div>
</div>

Pivot Middle On:

A child element calculates it's X values relative to it's pivot. In default it's the center of any element. But using the data-middle HTML5 attribute you can make the framework calculate the X value by taking the top left of the element as the pivot point. Here's how it's done, 
	<div class="background" data-scroll="parent">
    <div class="content-handle" data-scroll="child" data-middle="on" data-to="right"></div>
</div>

Pivot Middle Off:

	<div class="background" data-scroll="parent">
    <div class="content-handle" data-scroll="child" data-middle="off" data-to="right"></div>
</div>

CCSCROLL.JS REFERENCE


ATTRIBUTE REFERENCE

Attribute Description Example
data-scroll This attribute is used to identify the scrolling element and the associated parent of it. "parent" and "child"
data-to Represents the destination of the element scrolling. "left", "right" and "center"
data-to Represents the starting direction of the scrolling element. "left", and "right"
data-factor The reduction amount of scolling as a float percentage. "0.52", "0.1" or "1"
data-middle States whether the element must calculate the X pivotting from center or form top left. "on" or "off"
